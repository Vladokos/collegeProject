Один из главных инструментов в создании страниц - HTML.

HTML (Hyper Тext Markup Language) - это язык гипертекстовой разметки для создания интернет страниц, был создан в приблизительно в 1986—1991 годах, а опубликован в 1993.

В прошлом уроке Вы с ним не много поработали добавив заголовок ( <h1> заголовок </h1> ). HTML не ограничивается только одним заголовком в этом языке используются и множество других тегов такие как <header> </header>,<div> </div>,<footer> </footer>. . . но бояться нечего потому что в основном для создания страницы будут использоваться не все теги, а малое их количество.

Давайте разбираться.

В начале при создании HTML документа вам нужно будет написать его стандарт, на данный момент используется стандарт HTML5 и чтобы указать что мы его будем использовать нужно в начале написать <!DOCTYPE html> теперь браузер будет знать что мы используем стандарт HTML5

Далее нужно будет написать тег <html> </html> этот тег является контейнером для наших последующих тегов

Теперь нам нужно указать тег <head> </head> внутри этого тега будет содержаться информация для браузеров и поисковых систем. Внутри этого тега мы добавим еще два тега <meta charset="UTF-8">; и <title> Я заголовок </title>

<meta charset="UTF-8"> это тег определяет метатеги, которые используются для хранения информации предназначенной для браузеров и поисковых систем. В данный момент мы указали кодировку внутири тега UTF-8 для того чтобы мы могли писать, добавлять символы и это всё корректно отображалось на странце

<title> Я заголовок </title> этот тег задаёт заголовок для страницы. Когда Вы заходите на какую-нибудь страницу можно увидеть сверху что на вкладке, которую вы открыли есть название это задает <title> </title>

После тега <header> </header> нужно добавить тег <body> </body>внутри этого тега будет содержаться множество других тегов с данными
