Списки

В HTML существуют списки. Каждый список начинается с определенного тега <ul> </ul> или <ol> </ol>

Для того чтобы создать список надо использовать сочетание тегов <ul> <li> </li> </ul> - это маркированный список в нем каждый элемент списка будет отображаться с точкой.

<ol> <li> </li> </ol> - это нумерованный список в нем элементы будут отображаться с цифрой.

  • <ul> </ul> - означает что элементы внутри списка будут маркированные
  • <ol> </ol> - означает что элементы внутри списка будут отображаться цифрой
  • <li> </li> - элемент списка

Картинки

В HTML при помощи тегов можно добавлять картинки. Для того чтобы добавить картинку на сайт нужно использовать тег <img />

Для того чтобы добавить с помощью этого тега картинку на сайт нужно внутри него добавить путь к картинке <img src="путь к картинке" /> так же хорошим тоном будет добавить альтернативный текст для картинки. Делают это для того чтобы если картинка не смогла загрузиться, то это позволяет получить текстовую информацию о картинке. Делается это так <img src="путь к картинке" alt="текст"/>

    Путь указывается таким методом
  • ./ название файла .расширение - это если файл не находиться в папке
  • ./ название папки / название файла .расширение - файл который находиться в папке
    • ./ - означает что мы выходим из файла ../ - означает что мы выходим из текущей папки

Ссылки

Также по похожему методу можно добавлять ссылки на иные сайты, ресурсы и тому подобное. Для этого используют тег <a href="ссылка"> текст </a>

можно внутри ссылки поместить картинку будет выглядеть так <a href="ссылка"> <img src="путь к картинке" alt="текст"/> </a>

В данном случае пример ссылки с картинкой выступает две стрелочки снизу которые перенесут вас на следующий урок и одна стрелка сверху которая перенесет вас на главную страницу

Кнопки

На сайте могут присутствовать кнопки который буду выполнять некоторые действия на пример в интернет-магазинах есть кнопки, которые добавляют товар в корзину Для того чтобы добавить кнопку надо использовать этот тег <button> текст </button>

Также в этот тег можно добавить картинку и будет кнопка с картинкой <button> текст <img src="путь к картинке" alt="текст"/> </button>

Строка ввода

Иногда надо добавить строку ввода на сайт на пример для регистрации пользователя на сайте. Для этого используют тег <input type="text"/>

type - означает какой тип ввода будет, он может быть, как текстовое поле "text", поле с паролем "password", переключатель "radio", флажок "checkbox", скрытое поле "hidden", кнопка "button", кнопка для отправки формы "submit", кнопка для очистки формы "reset", поле для отправки файла "file" и кнопка с изображением "image"
