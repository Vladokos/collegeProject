Списки

В HTML существуют списки. Каждый список начинается с определнного тега <ul> </ul> или <ol> </ol>

Для того чтобы создать список надо использовать сочетание тегов <ul> <li> </li> </ul> - это маркированный список в нем каждый элемент списка будет отображаться с точкой.

<ol> <li> </li> </ol> - это нумерованный список в нем элементы будут отображаться с цифрой.

  • <ul> </ul> - означает что элементы внутри списка будут маркированные
  • <ol> </ol> - означет что элементы внутри списка будут отображаться цифрой
  • <li> </li> - элемент списка

Картинки

В HTML при помощи тегов можно добавлять картинки. Для того чтобы добавить картинку на сайт нужно использовать тег <img />

Для того чтобы добавить с помощью этого тега картинку на сайт нужно внутри него добавить путь к картинке <img src="путь к картинке" /> так же хорошим тоном будет добавить альтернативный текст для картинки. Делают это для того чтобы если картинка не смогла загрузиться то это позволяет получить текстовую информацию о картинке. Делается это так <img src="путь к картинке" alt="текст"/>

    Путь указывается таким методом
  • ./ название файла .расширение - это если файл не находиться в папке
  • ./ название папки / название файла .расширение - фалй который нахоидться в папке
    • ./ - означает что мы выходим из файла ../ - означает что мы выходим из текущей папки
