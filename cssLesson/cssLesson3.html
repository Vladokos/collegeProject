Редактирование текста

Иногда стандартные стили текста в HTML не подходят к стилю сайта и на помощь приходит опять CSS

В CSS существует множество свойств для редактирвоания текста от изменения его цвета до изменения его высоты

Ниже будут расписаны теги для изменении стилей текста

  • color: #hex - задаёт цвет текста.
    #hex - означает цифровой номер цвета например
  • font-size: число px; - задает размер шрифта
    font-size: 16px; - шрифт будет 16px, например этот шрифт 16px
  • font-weight: (100-900) - задает насыщенность шрифта, диаппазон от 100 до 900
  • line-height: число px - устанавливает межстрочный интервал для текста.
    для примера:
    Lorem Ipsum - это текст-"рыба", часто используемый в печати и вэб-дизайне. Lorem Ipsum является стандартной "рыбой" для текстов на латинице с начала XVI века.
  • font-family: имя шрифта - на сате используется данный шрифт font-family: "Montserrat";

Это были стили для текста которые используются в работе намного чаще чем остальные вы можете опробовать их создав заголовок или что-то другое после подключить стили и задавать значение вашему элементу и смотреть как все меняется
