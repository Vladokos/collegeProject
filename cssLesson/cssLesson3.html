Редактирование текста

Иногда стандартные стили текста в HTML не подходят к стилю сайта и на помощь приходит опять CSS

В CSS существует множество свойств для редактирвоания текста от изменения его цвета до изменения его высоты

НИже будут расписаны теги для изменении стилей текста

Разбор сделанного

В этой работе вы подклчюили стили CSS к HTML, но поскольку изначально в фалйе со стилями ничего не было нечего на сайте не изменилось, но после того как вы добавили заголовок первого уровня и в стилях задали ему цвет он стал зеленым

в CSS вы написали h1{color:green;} h1 - это селектор то у чего CSS будет менять стили селектором может быть и div,footer,heade и так далее

color - это свойство которые вы задаете данному селектору в данном случаем Вы изменили цвет у текста, но в CSS можно менять ширину(width), высоту(height) и так далее

green - это значение свойства, но у каждого свойства есть свои значение так что не получиться при изменении ширины задать цвет
