Позиционирование

В CSS позиционирование описывает, как любой из элементов может быть размещен независимо от порядка документа для этого есть несколько свойств и первое свойтсво это float

Float

Float - смещается при помощи обтекания влево или впрвао.
Пример:

Lorem ipsum dolor sit amet, consectetur adipiscing elit. Aliquam pharetra ante justo, a pretium leo pretium sit amet. Morbi fringilla turpis ut urna rhoncus, et mattis magna facilisis. Donec ut sapien luctus metus consequat imperdiet eu eget nulla. Pellentesque in nulla dictum, maximus neque ac, ornare lorem. Donec dui nibh, scelerisque eu dui ut, volutpat pulvinar dui. In tempor a risus vitae dapibus. Pellentesque odio nibh, tempus non viverra id, aliquet vitae leo. In gravida ex libero, vitae elementum dolor laoreet eget.

Понажимайте на белый блок и вы увидите как работает float

У свойства float есть 4 значения:

  • none - отсутствие обтекания
  • left - элемент будет слева
  • right - элемент будет справа
  • inherit - значение будет как родителя(элемента который находиться сверху другого - родитель)

Position

Position - задает способ позиционирования элемента относительно окна браузера или других объектов на веб-странице

У своства position есть 7 значений:

  • static - элементы отображаются как обычно
  • relative - положение элемента устанавливается относительно его исходного места. добавление свойств top, left,right и bottom меняет его расположение.
  • absolute - блок вынимаеться из потока документа(как-будто парит в воздухе) и становиться независим от изначальной позиции. добавление свойств top, left,right и bottom меняет его расположение.
  • sticky - делает блок липким который приклеивается к границе окна страницы и будет липким до того момента пока граница родителя не закончится
  • fixed - фиксированное позиционирование аналогично абсолютному позиционированию, с отличием в том, что он привязывается к указанной точке на экране свойствами left, top, right и bottom
  • initial - значение по умолчанию
  • inherit - наследует позиционирование от родителя

Для примера бургер меню(3 полоски на шоколадном фоне) использует абсолютное позиционирование( position: absolute;)

Position

Данные свойства используются вместе с position которое имет значение отличное от static без него они будут бесполезны

Чтобы задать смещение с помощью этих свойств надо написать
top или bottom или left или right: число px
также эти свойства можно использовать вместе для одного элемента

z-index

z-index - в CSS каждый блок имеет позицию в трех измерениях. В дополнение к горизонтальному и вертикальному положению, блоки выкладываются вдоль оси Z друг над другом.

Для того чтобы поменять расположение блока чтобы один был над другим надо написать свойтсво
z-index: число(от -∞ до +∞)
