Что такое Flexbox?

Flexbox - это технология для создания сложных гибких макетов за счёт правильного размещения элементов на странице.

Как использовать Flexbox?

Для того чтобы использовать Flexbox в css в классе нужно приписать свойство display:flex;

Нажмите на любой из 3 кубиков или рядом с ними чтобы увидеть результат

Текущее состояние:
display:block;
