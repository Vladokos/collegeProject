Для чего нужны классы?

До этого момента Вы задавали стили с помощью названия тега, но на сайте теги могут повторяться (например, на сайте часто используют больше двух div) и для того чтобы каждому из тегово задать свой стиль нужно использовать классы

Как использовать классы?

Для того чтобы использовать атрибут класс нужно внутри тега написать class="что внутри будет содержать блок"

Например <div class="theoryClass">...</div>
это означает что теперь у данного блока есть класс который называется theoryClass теперь в CSS можно будет писать не div{...}, а .theoryClass{..}

Для того чтобы обратиться к классу в CSS нужно поставить точку после точки написать название класса и в фигурных скобочках можно писать стили
.название__класса{стили}

Также стоит помнить, что задавать название класса нужно на английском языке и осознанно, то есть когда вы будете задавать название класса вы должны как будто вкратце описывать что там будет

например, нужно создать блок внутри которого будут товары тогда этот класс нужно назвать goods(товары)
или нужно создать блок внутри которого будут определённые типы товаров допустим чай то класс будет называться goodsTeas(товары чаи) этот класс в вкратце описывает что внутри будут товары категории чай

Классы для шапки и подвала сайта буду называться также, как и тег class="header" и class="footer"

Задание

Попробуйте внизу создать два блока сайта внутри них написать какой-нибудь текст и задать им разные классы, а после в CSS попробовать задать им стили по классам

И не забудьте подключить стили
