Ширина и высота

Если вдруг вам понадобиться изменить ширину и/или высоту текста, изображения или чего-то ещё то тут приходят на помощь свойства width и height

Значения для свойства width и height задаются таким образом width: значение px ; height: значение px;

отступы

Иногда разные блоки на сайте стоят слишком близко друг к друг и для того чтобы их отталкнуть используют отступы

Есть два типа отступа padding и margin

padding: число px(верхний отступ) число px(отступ справа) число px(нижний отступ) число px(отступ слева) - задает внутренние отсупы увеличивая его границы.

Для того чтобы легче было запомнить с какой стороны какой отступ есть один способ значения идут по часовой стрелке то есть верхний отступ - 12 часов, отступ справа - 3 часа, нижний отступ - 6 часов, отступ слева - 9 часов.

margin: число px(верхний отступ) число px(отступ справа) число px(нижний отступ) число px(отступ слева) - задает внешние значение отталкивая контент и смещаю в одну из сторон от другого контента
