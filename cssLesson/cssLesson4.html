Ширина и высота

Если вдруг вам понадобиться изменить ширину и/или высоту текста, изображения или чего-то ещё то тут приходят на помощь свойства width и height

Значения для свойства width и height задаются таким образом width: значение px ; height: значение px;

Отступы

Иногда разные блоки на сайте стоят слишком близко друг к друг и для того чтобы их оттолкнуть используют отступы

Есть два типа отступа padding и margin

padding: число px(верхний отступ) число px(отступ справа) число px(нижний отступ) число px(отступ слева) - задает внутренние отступы увеличивая его границы.

Для того чтобы легче было запомнить с какой стороны какой отступ есть один способ значения идут по часовой стрелке, то есть верхний отступ - 12 часов, отступ справа - 3 часа, нижний отступ - 6 часов, отступ слева - 9 часов.

margin: число px(верхний отступ) число px(отступ справа) число px(нижний отступ) число px(отступ слева) - задает внешние значение отталкивая контент и смещаю в одну из сторон от другого контента

Задний фон

Бывают случаи, когда надо изменить фон для этого создали селектор backgound, а точнее backgound-color

Селектор backgound оно не одно у него есть множество разновидностей например color,image,size,position,repeat

Давайте разберёмся для чего какой подходит

  • backgound-color: #hex - задет цвет заднему фону
  • backgound-image: url("ссылка на изображение") - ставит на задний фон картинку
  • backgound-size: contain- занимает весь блок/ cover - масштабирует с сохранением пропорций
  • backgound-position: top,bottom, left, right, center - устанавливает начальную позицию картинки
  • backgound-repeat: repeat, space no-repeat - будет ли повторяться изображение

Вот пример:
background-image:url("ссылка на картинку");
background-size: contain;
background-repeat: no-repeat;

Второй пример:
backgound-color: olive;
