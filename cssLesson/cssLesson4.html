Ширина и высота

Если вдруг вам понадобиться изменить ширину и/или высоту текста, изображения или чего-то ещё то тут приходят на помощь свойства width и height

Значения для свойства width и height задаются таким образом width: значение px ; height: значение px;

Отступы

Иногда разные блоки на сайте стоят слишком близко друг к друг и для того чтобы их отталкнуть используют отступы

Есть два типа отступа padding и margin

padding: число px(верхний отступ) число px(отступ справа) число px(нижний отступ) число px(отступ слева) - задает внутренние отсупы увеличивая его границы.

Для того чтобы легче было запомнить с какой стороны какой отступ есть один способ значения идут по часовой стрелке то есть верхний отступ - 12 часов, отступ справа - 3 часа, нижний отступ - 6 часов, отступ слева - 9 часов.

margin: число px(верхний отступ) число px(отступ справа) число px(нижний отступ) число px(отступ слева) - задает внешние значение отталкивая контент и смещаю в одну из сторон от другого контента

Задний фон

Бывают случаии когда надо изменить фон для этого создали селектор backgound, а точнее backgound-color

Селектор backgound оно не одно у него есть множетсво разновидностей например color,image,size,position,repeat

Давайте разебремся для чего какой подходит

  • backgound-color: #hex - задет цвет заднему фону
  • backgound-image: url("ссылка на изображение") - ставит на задний фон картинку
  • backgound-size: auto(автоматичиски маштабирует картинку)/ contain(масштабирует картинку так, чтобы она максимально накрыла собой весь блок)/ cover(масштабирует изображение как можно больше c сохранением пропорций изображения) - позволяет задавать размер картинки
  • backgound-position: top(изображение будет сверху),bottom(изображение будет снизу), left(изображение будет слева), right(изображение будет справ), center(изображение будет в центре) - устанавливает начальную позицию картинки
  • backgound-repeat: repeat(изоброжение будет повторяться), space(изображение будет повторяться с отступами друг от друга) no-repeat(без повторений) - будет ли повторяться изображение для заполнение пустого пространства
  • backgound: свойства - можно использовать один Селектор для задания нескольких свойств(размера,позиции)

Вот пример background-image:url("ссылка на картинку") + background-size: contain; background-repeat: no-repeat; и backgound-color: olive;
