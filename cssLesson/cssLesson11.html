Анимации что это и как ?

В css можно задавать анимацию 2 спсобами через transition и @keyframes

transition

transition - задает анимацию при изменении свойств

Для примера навидите на квадрат

Для того чтобы это анимировать всего лишь понадобилось задать стили тегу
.translitionBlockExample{ background-color: #3f2f22; width: 150px; height: 150px; transition: 0.8s ease-in; }
тут были заданы цвет заднего фона для блока, ширина, высота и translition
translition: 0.8s - это длительность анимации, а ease-in - это ее плавноость

На этом сайте вы можете посмотреть более обширный список анимаций с помщью translation

Далее добавлен был псевдокласс :hover и стили
.translitionBlockExample:hover{ width: 90px; height: 90px; background-color: #a7622a; }
При наведении эти стили применяются но с анимацией и после потери фокуса стили возвращаются к изначальному состоянию также с анимацией

keyframes

@keyframes - ключевые кадры используются для указания значений свойств анимации в различных точках анимации. Ключевые кадры определяют поведение одного цикла анимации; анимация может повторяться ноль или более раз.

Чтобы задать keyframes нужно использоть эту структуру:
@keyframes имя анимации { список правил }

Список правил - это установка ключевых кадров.
Например:

@keyframes shadow { from {text-shadow: 0 0 3px black;} 50% {text-shadow: 0 0 30px black;} to {text-shadow: 0 0 3px black;} }

from - первый кадр, аналогичен 0%

50% - на середине анимации

to - последний кадр, аналогичен 100%.

Наведите на текст чтобы анимация запустилась

Lorem, ipsum dolor sit amet consectetur adipisicing elit. Esse nihil alias a fugit deleniti rem illum, nisi ad ea dicta porro suscipit ex quo asperiores quos eos id ipsum sed.

После того как были созданы @keyframes, надо добавить анимацию тегу для этого нужно внутри тега написать:

animation: имя анимаций длительность анимации кол-во повторений функция плавности

  • Длительность - задается в секундах 2s,0.1s и так далее
  • кол-во повторений - infinite(бесконечное) или в числах 0,1,2,3
  • функция плавности - ease-in-out, ease-in и так далее
