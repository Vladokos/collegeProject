Что такое Flexbox?

Flexbox - это технология для создания сложных гибких макетов за счёт правильного размещения элементов на странице.

Как использовать Flexbox?

Для того чтобы использовать Flexbox в css в классе нужно приписать свойство display:flex;

Нажмите на любой из 3 кубиков или рядом с ними чтобы увидеть результат

Текущее состояние:
display:block;

flex-direction

У flex-контейнера есть две оси: главная и перпендикулярная ей. По умолчанию все предметы располагаются вдоль главной оси — слева направо. Именно поэтому блоки в предыдущем примере выстроились в линию, когда мы применили display: flex. А вот flex-direction позволяет вращать главную ось.

У свойства flex-direction елсть несколько значенией

  • flex-direction: row; - сдвигает все элементы в строку
  • flex-direction: column; - сдвигает все элементы в столбец
  • flex-direction: row-reverse; - переворачивает строку
  • flex-direction: column-reverse; - переворачивает столбец

Примеры:

Текущее состояние:
flex-direction: row;

justify-content

justify-content - отвечает за выравнивание элементов по главной оси

У свойства justify-content есть несколько значенией

  • flex-start - распологает элементы в начале строки
  • flex-end - распологает элементы в конце строки
  • center - распологает элементы по центру строки
  • space-between - элементы равномерно распределяются по всей строке.Первый и последний элемент прижимаются к соответствующим краям контейнера.
  • space-around - элементы равномерно распределяются по всей строке. Пустое пространство перед первым и после последнего элементов равно половине пространства между двумя соседними элементами.

Примеры:

Текущее состояние:
justify-content: flex-start;

align-items

align-items - Если justify-content работает с главной осью, то align-items работает с осью, перпендикулярной главной оси.

У свойства align-items есть 5 значенией

  • flex-start - элементы выравниваются относительно верхнего края (начала) контейнера.
  • flex-end - элементы выравниваются относительно нижнего края (конца) контейнера.
  • center - элементы выравниваются по центру контейнера.
  • stretch - элементы растягиваются, занимая все пространство контейнера по высоте.
  • baseline - элементы выравниваются по базовой линии.

Текущее состояние:
align-items: flex-start;
