Что такое псевдоклассы?

Псевдокласс в CSS — это ключевое слово, добавленное к селектору, которое определяет его особое состояние.

Как использовать Псевдоклассы?

Все псевдоклассы задаются одинаково и для того чтобы использовать псевдокласс вам надо к класу приписать псевдокласс

Например:
.название_класса:псевдокласс{ свойства } .example:hover{ color:#3F2F22; }

Наведите на текст чтобы увидеть результат

Текущее состояние:
color: #f0e7d6;

При наведения указателя мыши на этот текст срабавтывает псевдокласс :hover и заменяет цвет текста на коричневый

:hover

:hover - псевдокласс срабатывает, когда пользователь наводит на элемент мышью, но не обязательно активирует его.

Этот псевдокласс можно использовать с другимми псевдоклассами такими как :link, :visited и :active

:link

:link - позволяет задавать стили не посещенных ссылок.

Также этот псевдокласс можно использовать с другими такими как :hover, :active или :visited.

Для примера не посещенной ссылке ниже изменили тип цвета, но если ее посетить то она станет другого цвета

:visited

:visited - позволяет задавать стили только тем ссылкам которые были посещены.

Также этот псевдокласс можно использовать с другими такими как :link, :hover и :active

Для примера поситите следующую ссылку и посмотрите как изменится ее цвет после посещения

Тестовая ссылка
:active

:active - позволяет задавать стили только тем который активирует пользователь.

Также этот псевдокласс можно использовать с другими такими как :link, :hover и :active
