Что такое CSS?

Если вам вдруг не нравяться цвет текста в HTML или вам хочется поменять задний фон то для этого придумали CSS

CSS (Cascading Style Sheets) или же каскадные таблицы стилей. Впервы о них упомянул Хокон Виум Ли в 1994 году, а были приняты W3C 17 декабря 1996.

Как начать с ним работать

Для того чтобы совместить работу HTML и CSS В HTML нужно прописать данный тег <link rel="stylesheet" href="ссылка на CSS"> после тега <meta charset="UTF-8" />

rel="stylesheet" - означает что мы добавляем таблицы стилей на сайт

После того как был добавлен CSS на сайт можно переходить к настройки стилей в CSS файле и они будут применятся в HTML
