While

Цикл while имеет следующий синтаксис:

while (условие) {
код
}

Код будет выполняется, пока условие истинно.

Например, цикл ниже выводит i, пока i <= 3 :

let i = 0;
while (i <= 3) {
console.log( i );
i++;
}

В результате в консоль выведется 0,1,2,3

do...while

Цикл do...while имеет следующий синтаксис:

do {
код
} while (condition);

Цикл do...while выполниться один раз вне зависимости от условия, например:

let i = 0;
do {
console.log( i );
i++;
} while (i < 3);

for

Более сложный, но при этом самый распространённый цикл — цикл for.

Выглядит он так:

for (начало; условие; шаг) {
код
}

Давайте разберёмся, что означает каждая часть, на примере. Цикл ниже выполняет console.log(i) для i от 0 до (но не включая) 3:

for (let i = 0; i < 3; i++) {
console.log(i);
}

В результате выведется 0,1,2

Рассмотрим конструкцию for подробней:

  • Начало let i = 0 - объявлется переменная с начальным значением(обязательно)
  • Условие i < 3 - ставиться условие до каких пор будет выполняться цикл. Проверяется при каждом шаге(выполнение кода)
  • Шаг i++ - шаг выполняется полсе кода, каждый шаг перед проверкой условия
