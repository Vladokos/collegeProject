Объявление функции

Для создания функций мы можем использовать объявление функции.

Пример объявления функции:

function имя(параметры) {
тело(код)
}

После объявления функции надо её вызвать чтобы её тело сработало, чтобы это сделать надо написать её имя и если нужно, то передать параметры:

имя(параметры);

Пример:

function showMessage() {
console.log( 'Всем привет!' );
}
showMessage();

В результате будет выведено в консоль 'Всем привет!'

параметры - это значение которое будет принимать в себя функция и использовать внутри своего тела, например:

function test(n){
console.log(n);
}
test(2);

Тут есть функция test, которая внутрь себя принимает параметр с названием n, далее вызывается функция test внутрь которой передается 2, n принимает в себя два и становится n = 2, а после в console.log() выводиться её значение
