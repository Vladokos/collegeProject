Подготовка

Для того чтобы JS начал работать его нужно подключить в HTML документе похожим образом как CSS

Перед закрывающим тегом body нужно добавить ссылку на JS

<script src="./имя или путь к файлу .js"> </script>

После этого можно приступать к написанию кода в файле JS

JS подключают снизу из-за того что этот файл может содержать команды который будут работать только после загрузки HTML и CSS

Hello world!

Первая программа будет - поздороваться с миром, это такая традиция при изучении нового языка программирования нужно поздороваться с миром

Для того чтобы это сделать нужно будет:

  • Подключить JS к HTML фалу
  • в JS файле написать console.log("Hello world!");
  • После этого надо открыть "Console" или нажмите на F12(ПКМ по пустому месту и консоль разработчика или просмотреть код) и в консоле будет вывод текста

console.log() - эта команда выводит в консоль то что будет в скобках

Так же есть команда, которая работает как предупреждение - alert()

alert() - выводит окно сверху браузера(зачастую используется как предупреждение) и показывает данные из скобок
