addEventListener

элемент.addEventListener(событие, функция)

Пример:

let doc = document.getElementById("example");
doc.addEventListener("click", () => {
console.log("Click!");
});

Получаем тег по id после добавляем ему реакцию на событие в данном случае используется реакция на нажатие. В итоге все выводиться в консоль

События

  • click – происходит, когда кликнули на элемент левой кнопкой мыши (на устройствах с сенсорными экранами оно происходит при касании)
  • contextmenu – происходит, когда кликнули на элемент правой кнопкой мыши
  • mouseover / mouseout – когда мышь наводится на / покидает элемент
  • mousedown / mouseup – когда нажали / отжали кнопку мыши на элементе
  • dmousemove – при движении мыши
  • submit – пользователь отправил форму <form>
  • focus – пользователь фокусируется на элементе, например, нажимает на <input>
  • keydown и keyup – когда пользователь нажимает / отпускает клавишу

Задание

Подключите JS к HTML и создайте один div внутри него какой-нибудь контент, после добавьте в JS addEventListener на этот div и чтобы когда мышь будет на элементе в консоль выводилось "Он тут"
