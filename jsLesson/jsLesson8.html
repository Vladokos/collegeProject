addEventListener

элемент.addEventListener(событие, функция)

Пример:

let doc = document.getElementById("example");
doc.addEventListener("click", () => {
console.log("Click!");
});

Получаем тег по id после добавлаем ему реакцию на событие в данном случае используется реакция на нажатие

События

  • document.getElementById() - получает данные тега по его id
  • document.querySelectorAll() - возвращает все элементы которые подходят условию
  • document.querySelector() - возвращает первый элемент который подходит условию
  • document.getElementsByTagName() - ищет элементы с данным тегом и возвращает их значения
  • document.getElementsByClassName() - возвращает элементы, которые имеют данный класс.
  • document.getElementsByName() возвращает элементы с заданным атрибутом name.

document.getElementById(), пример:

let doc = document.getElementById("example");
console.log(doc);

В данном примере doc получает данные тега по id example а после выводит эти данные в консоль

Подключите JS к HTML и создайте один div задайте ему id, а в JS получити этот div по id и вывидите его данные в консоль
