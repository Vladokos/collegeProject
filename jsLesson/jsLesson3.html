Переменные

В JavaScipt есть 3 переменных

  • var - объявляет переменную с глобальной видимостью
  • let - объявляет переменную с локальной видимостью
  • const - объявляет константу(переменная которой можно присвоить значение а дальше его изменять нельзя)

Пока для начала в большенстве случаев лучше использовать let

Для того чтобы присвоить значение переменной нужно написать переменную, название и значаение, пример:

let(переменная) name(название) = 10(значение);

После этого название переменной будет хранить значение, которое можно будет использовать в других действиях. Для примера console.log(name) выведет цифру 10 так как name хранит в себе 10 (name = 10)

Типы данных

В JavaScipt есть 8 типов данных:

  • String - строка
  • Number - число
  • BigInt - предназначен для очень больших целых чисел
  • Boolean - логическое значение true или false
  • Undefined - указывает, что значение не установлено
  • Null - указывает на отсутствие значения
  • Symbol - используется для создания уникальных идентификаторов в объектах
  • Object - представляет комплексный объект
String

String - строка, содержит в себе слова. Для того чтобы задать сроку надо после равно написать двойные или одинарные кавычки, например:

let greg = "Greg";

Здесь создана переменная let с её названием greg, которая содержит в себе строку(текст) "Greg"

Number и BigInt

Number, BigInt - содержат в себе разные виды чисел как десятичные(числа с плавающей запятой) так и целые. Отличие Number от BigInt в том что BigInt вмещает в себя более длинные значения чем Number. Для того чтобы задать Number или BigInt надо после равно написать любое значение, например:

let number = 1234;

let dropNumber = 12.1;

Здесь создана переменная let с её названием number, которая содержит в себе число 1234

Во второй строке была создана переменная с плавающей запятой(десятичное)

Boolean

Boolean - логический тип данных, который представляет true(истина) и false(ложь). В основном используется для сравнения, например:

console.log(12 > 3);

Здесь сравнивается два числа 12 и 3 и условие сравнение это что 12 больше 3? В результате сравнение в консоль выведет true так как 12 > 3

Для того чтобы объявить переменную с типом данных Boolean, нужно:

let logical = false;

Здесь создана переменная let с её названием logical, которая содержит в себе false

Null и Undefined

Null, Undefined - указывает, что значение не определено или не установлено. В основном будет выводиться в консоли в ходе неправильных действий

Object

Object - проще говоря объект это ящик с подписанными папками. Каждый элемент данных хранится в своей папке, на которой написан ключ. По ключу папку легко найти, удалить или добавить в неё что-либо. например:

let obj = {name: "Tom", age: 32};

Здесь создана объект, который содержит в себе 2 папки и их ключи это name и age, а элементы данных "Tom" и 32

Symbol

Symbol - это уникальный и неизменяемый тип данных, который может быть использован как идентификатор для свойств объектов.
