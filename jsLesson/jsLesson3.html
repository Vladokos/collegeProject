Переменные

В JavaScipt есть 3 переменных

  • var - объявляет переменную с глобальной видимостью
  • let - объявляет переменную с локальной видимостью
  • const - объявляет константу(переменная которой можно присвоить значение а дальше его изменять нельзя)

Пока для начала в большенстве случаев лучше использовать let

Для того чтобы присвоить значение переменной нужно написать переменную, название и значаение, пример:

let(переменная) name(название) = 10(значение);

После этого название переменной будет хранить значение, которое можно будет использовать в других действиях. Для примера console.log(name) выведет цифру 10 так как name хранит в себе 10 (name = 10)

Типы данных

В JavaScipt есть 8 типов данных:

  • String - строка
  • Number - число
  • BigInt - предназначен для очень больших целых чисел
  • Boolean - логическое значение true или false
  • Undefined - указывает, что значение не установлено
  • Null - указывает на отсутствие значения
  • Symbol - используется для создания уникальных идентификаторов в объектах
  • Object - представляет комплексный объект
