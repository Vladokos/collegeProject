Конструкция if...else...else if

Для того чтобы сравнить некоторые данные и выполнить потом действия используется if, например:

if(12 < 22) {
console.log("12 меньше 22");
}

Здесь задается условие if(если) 12 меньше 22 то выполняется то что написано в фигурных скобках, в данном случае console.log("12 меньше 22");

Если вдруг для примера 12 будет больше 22, то тогда что внутри фигурных скобка не выполниться, а просто пропустит

else - используется для того что если if будет ложно то выполниться else, например

if(12 > 22){
console.log("12 больше 22");
}else{
console.log("12 меньше 22");
}

Здесь есть условие if, что если 12 больше 22, то выполниться, то что в фигурных скобка, а если это неправда что 12 больше 22 то тогда выполниться else

else if - используется если условие if ложно, но нужно проверить другое условие, например:

if(12 > 22){
console.log("12 больше 22");
}else if (12 < 22){
console.log("12 меньше 22");
}else{
console.log("они равны");
}

Здесь если условие if (12 > 22) ложно, то проверяется условие else if(иначе если) 12 меньше 22, то выполниться то что внутри фигурных скобках, а если и else if ложно, то выполниться else

Операторы сравнения

Многие операторы сравнения известны нам из математики

В JavaScript они записываются так:

  • Больше/меньше: a > b, a < b
  • Больше/меньше или равно: a >= b, a <= b
  • Равно: a == b. Обратите внимание, для сравнения используется двойной знак равенства ==. Один знак равенства a = b означал бы присваивание.
  • Не равно. В математике обозначается символом ≠, но в JavaScript записывается как a != b.

Результат сравнений возвращает логический тип true(истина) или false(ложь)

Логические операторы

В JavaScript есть логические операторы, которые призваны помочь вам в сравнении каких-либо значений, есть 3 логических оператора:

  • ИЛИ || - обозначается двумя вертикальными полосами
  • И && - обозначается как два амперсанда
  • НЕ ! - обозначается как восклицательный знак

Пример с || (Логическое ИЛИ):

let hour = 2;
if (hour < 10 || hour > 18){
console.log("Закрыто");
}

Здесь создали переменную hour со значение 2 и в операторе if стоит условие что если hour меньше 10 ИЛИ hour больше 18 то выполниться действие внутри фигурных скобках

Пример с && (Логическое И):

let hour = 12;
let minutes = 10;
if(hour == 12 && minutes == 10){
console.log("Сейчас 12:10");
}

Здесь было создано две переменный hour и minutes со значением 12 и 10, после в операторе if происходит сравнение что если hour равен 12 И minutes равен 10, то выполниться действие внутри фигурных скобках

Пример с ! (Логическое НЕ):

let seconds = 12;
if(secons != 60){
console.log("Меньше 60");
}

Здесь было создана переменная seconds со значением 12 и оператор if который сравнивает, что если seconds НЕ РАВЕН 60 то выполняется действие в фигурных скобках
